\# Kyoto Sentiment Engine 🏯 (京都店舗・観光レビュー感情分析エンジン)



A Japanese NLP Sentiment Analysis tool built for Kyoto smart-commerce.  

FastAPIとJanomeを使用した、京都のスマートコマース向け感情分析ツールです。



\---



\## 🌟 Features (概要)

\- \*\*Custom-trained Model\*\*: Uses Naive Bayes to classify customer feelings.  

&#x20; (Naive Bayesアルゴリズムを使用した独自の感情分析モデル)

\- \*\*Japanese NLP\*\*: Advanced tokenization using Janome to focus on nouns and adjectives.  

&#x20; (Janomeを使用して名詞・形容詞・動詞を抽出。日本語の文脈に特化)

\- \*\*Web Interface\*\*: Real-time analysis via a clean browser UI.  

&#x20; (ブラウザからリアルタイムでレビューを分析できるUIを搭載)



\## 🛠️ Usage (使い方)

1\. Activate venv \& Run: `python main.py`

2\. Open `http://127.0.0.1:8000`

3\. Type a review and see the AI magic!
